# BetterDefect v0.10.0

适用版本：

- 手机：Slay the Spire 2 v0.103.x
- 电脑：Slay the Spire 2 v0.107.1
- 不需要 BaseLib

## 本次更新

百科大全的“改造”系统新增16张机器人蓝卡，继续与禁用卡牌共享35点上限，选择结果会持久保存：

- 混沌、双倍能量、强撑、快速检索
- 暴风雨、白噪声、超越光速、空值、折射
- 野性、冰雹风暴、迭代、循环、烟囱、雷暴、子程序

所有卡牌均同步了基础牌、普通升级牌、实际战斗效果和中文卡面描述。具体数值以游戏百科大全中的改造预览为准。

## 安装

解压后将 `BetterDefect` 文件夹放入游戏的 `mods` 目录。更新时直接覆盖旧版同名文件夹，已有的禁用、改造、动态出率和卡面选择记录会继续读取。

- Android v0.103.2 Release 编译通过。
