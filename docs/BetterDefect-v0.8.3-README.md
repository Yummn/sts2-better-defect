# BetterDefect v0.8.3

适配：手机版 v0.103.2、电脑版 v0.107.1；不依赖 BaseLib。

## 本版修复

- 修复局内查看牌组时错误显示“禁用出率”“改造”按钮、动态出率文字和禁用遮罩的问题。
- 原因一是游戏的局内主牌组和百科大全共用了 `NCardLibraryGrid` 类型；旧实现只判断网格类型，因此把百科大全 UI 注入了局内牌组。
- 原因二是通用的 `NGridCardHolder.SetIsPreviewingUpgrade` 延迟刷新也错误地被当成百科专用流程，会在第一次清理后再次塞回按钮。
- 现在网格分配、延迟刷新和升级预览刷新都会验证节点确实隶属于可见的 `NCardLibrary`；否则主动清除 BetterDefect 添加的按钮、动态出率文字和遮罩。
- 百科大全中的禁用、动态出率、历史版本改造及 35 点进度条保持不变。
- 保留 v0.8.2 的 Android 启动保护和此前所有卡牌效果修复。

## 真机验证

- REDMI K80 Pro、游戏 v0.103.2、全部 8 个模组同时启用。
- 先进入机器人百科大全，确认禁用、历史改造、动态出率和 35 点进度条仍正常显示。
- 再返回当前存档并打开局内主牌组，确认禁用按钮、历史改造按钮、动态出率文字和禁用遮罩均不再出现。
- 启动及操作日志未出现 BetterDefect 相关异常；测试前主存档和备份均已按 SHA-256 恢复。

## 安装

- 手机版 v103：`BetterDefect-v0.8.3.zip`
- 电脑版 v107.1：`BetterDefect-v0.8.3-PC-v107.1.zip`
- 解压或导入后，把 `BetterDefect` 文件夹放入游戏 `mods/`。

覆盖更新不会删除卡图选择、禁用状态、动态出率或历史版本改造记录。
