# BetterDefect v0.9.1

适用版本：

- PC：杀戮尖塔 2 v0.107.1
- 手机：杀戮尖塔 2 v0.103.2
- 不依赖 BaseLib

## 本次修复

- **Tesla Coil / 特斯拉线圈**：v0.105 改造启用时，描述明确显示“触发所有闪电球被动一次（两次）”；普通卡触发一次，卡牌自身升级后触发两次，与实际效果一致。
- **Shatter / 打碎**：描述修正为“激发所有充能球两次”，与实际逐球双重激发一致。
- **Reprogram / 重编程**：修复额外发现的升级效果错误。升级后仍只失去 1 点集中，力量与敏捷由 1 提升为 2，不再错误地失去 2 点集中。

## 一致性检查

- 静态检查 22 张 BetterDefect 重建卡牌及 25 条历史版本/自定义改造效果路由。
- 电动力学保持对所有敌人生效；裂变、回收、锁定、静电释放等既有修复保持有效。
- 未发现其他卡牌描述与实际行为不一致。

## 验证

- PC v0.107.1 与手机 v0.103.2 均编译通过，0 errors。
- 离线源代码及双平台二进制检查：96/96 通过。
- PC v0.107.1 实机启动成功：26 张旧卡注入，机器人卡牌总数 114，所有 BetterDefect 补丁加载完成，无运行时错误。
- 百科大全实机确认：特斯拉线圈 v0.105 改造描述为“一次（两次）”；打碎放大卡面显示“激发所有充能球两次”。测试后已将特斯拉线圈改造恢复为关闭。
