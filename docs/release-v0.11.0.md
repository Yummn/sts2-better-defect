# BetterDefect v0.11.0

## 新增的金卡改造

- 适应打击：2费，16(22)伤害；向抽牌堆加入一张0费、虚无的复制品。
- 万物一心：1费，6(9)伤害；从弃牌堆选择至多2(3)张当前为0费的牌加入手牌。
- 缓冲：2费，获得1(2)层缓冲和10点格挡。
- 吞噬暗影：生成1(2)个黑暗球；回合结束时触发所有黑暗球被动一次。
- 冷却剂：回合结束时，每有一种不同充能球获得2(3)格挡。
- 创造性 AI：每回合从3张随机机器人能力牌中选择1张，本回合免费。
- 回响形态：每回合第2张牌额外打出一次；升级移除虚无。
- 散射炮：消耗所有未被消耗的状态牌；按消耗牌堆总牌数对指定敌人造成6(9)伤害。
- 陨石打击：4费，18(24)伤害，生成2个等离子球。
- 多重释放：激发X(X+1)个充能球，并在每次激发后生成相同类型的充能球。
- 彩虹：3费，依次生成闪电、冰霜、黑暗、玻璃、等离子球；基础消耗，升级移除消耗。

## 稀有度与其他改造

- 遗传算法、冰之长枪、碎片整理：启用改造后变为蓝卡。
- 偏差认知：启用改造后变为金卡。
- 雷霆打击：效果不变，升级后改为2费。
- 核心电涌：效果不变，升级后获得固有。
- 稀有度变化时，已保存动态出率会迁移到新的稀有度池，并继续独立零和计算。
