# BetterDefect v0.8.1

适配：手机版 v0.103.2、电脑版 v0.107.1；不依赖 BaseLib。

## 卡牌描述一致性修复

- **火箭飞拳**：关闭 v0.100 历史改造时，描述现在明确写明 0 费只持续到“打出或当前回合结束”；启用改造时仍显示“下一次打出前”。
- **特斯拉线圈**：关闭 v0.105 改造时，升级牌不再错误显示闪电触发两次；启用改造后才显示升级牌触发两次。
- **燃料 / 压缩**：压缩使用 v0.108 基础行为时，燃料只显示获得能量，不再显示“抽 0 张牌”；启用 v0.99 改造后显示实际的抽 1/2 张牌。
- **刮削**：描述分别标明基础版本按卡牌自身耗能判断、v0.108 改造按当前最终耗能判断。
- **裂变**：普通牌显示“移除所有充能球”，升级牌显示“激发所有充能球”，不再同时显示两套行为。
- **核心电涌、裂变**：移除手写的重复“消耗”行，保留游戏根据真实 `Exhaust` 关键词自动生成的提示。
- **增幅**：实际效果现在与“本回合”描述一致；剩余层数会在玩家回合结束时移除，且每张能力牌通过原生 `AfterModifyingCardPlayCount` 路径消耗一层。

## 审计与验证

- 逐项审计 26 张恢复卡牌及 14 项历史版本改造。
- 自动化离线审计：66 项通过，0 项失败。
- PC v107.1 与手机版 v103 分别编译：均为 0 错误。
- 两份 DLL 均反编译确认包含四组动态描述、裂变条件描述及增幅回合结束清除逻辑。
- 按用户要求继续离线验证，未启动或覆盖 Steam 游戏目录。

## 安装

- 手机版 v103：`BetterDefect-v0.8.1.zip`
- 电脑版 v107.1：`BetterDefect-v0.8.1-PC-v107.1.zip`
- 解压或导入后，把 `BetterDefect` 文件夹放入游戏 `mods/`。

ZIP 只包含使用 `/` 的文件条目且不写目录条目，兼容手机版设置页导入器。配置保存在独立用户数据目录，替换 MOD 不会重置卡图、禁用状态、动态出率或历史版本选择。
