## BetterDefect v0.8.0

本版对恢复的一代故障机器人卡牌进行了完整离线审计。

### 修复

- 电动力学：闪电球被动与激发现在都会命中全部敌人。
- 回收：恢复手牌选择；X 费牌按当前剩余能量返还。
- 锁定：修正为 1.5 倍乘数，并在每个敌人回合结束时减少 1 层，不再按命中次数消耗。
- 静电释放：只响应未被格挡的攻击伤害。

### 验证

- 26 张恢复卡牌注册与 22 张重建卡牌效果/数值路由完成离线审计。
- 14 张历史版本升级映射与关键行为补丁完成复核。
- PC v107.1 与 Android v103 分别编译成功，均为 0 错误。
- 两份 DLL 反编译复核通过；自动化检查 59/59 通过。
- 按用户要求，本轮未启动或控制 Steam 游戏。详细结果见随 Release 上传的两个审计报告。
