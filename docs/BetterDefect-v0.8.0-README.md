# BetterDefect v0.8.0

适配：手机版 v0.103.2、电脑版当前 v0.107.1；不依赖 BaseLib。

## 本次修复

- **电动力学**：改为补丁接管闪电球被动与激发共用的 `ApplyLightningDamage` 路径，两种触发都会同时命中全部可攻击敌人，不再只有单体被动伤害。
- **回收**：恢复一代的手牌选择界面，不再自动消耗最左侧卡牌；X 费牌按打出回收后剩余的当前能量值返还能量。
- **锁定 / 瞄准靶心**：伤害修正返回正确的 `1.5x` 乘数，不再把伤害值再次当乘数；层数改为每个敌人回合结束时减少 1，不再被每次充能球命中消耗。
- **静电释放**：只在受到未被格挡的攻击/怪物招式伤害时触发，不再被中毒、遗物、充能球或其他生命流失错误触发。

## 离线审计结果

- 恢复卡池：26 张（游戏内隐藏 4 张 + 重建的一代卡牌 22 张）。
- 历史版本升级：14 张注册、数值路由和行为补丁全部通过源码检查。
- 电脑版与手机版 v103 分别编译：均为 0 错误。
- 两份 DLL 均反编译复核了电动力学共用路径、回收选择/X 费、锁定倍率/回合计时和静电释放过滤逻辑。
- 自动化离线审计：59 项通过，0 项失败。

本次按用户要求未启动或控制 Steam 游戏。`OFFLINE-AUDIT` 与 `BINARY-AUDIT` 文件记录的是源码、编译和反编译验证；下一次可在不占用 Steam 时补做实战回归。

## 安装

- 手机版 v103：使用 `BetterDefect-v0.8.0.zip`，导入或把其中 `BetterDefect` 文件夹复制到游戏 `mods/`。
- 电脑版 v107.1：使用 `BetterDefect-v0.8.0-PC-v107.1.zip`，把其中 `BetterDefect` 文件夹复制到游戏 `mods/`。
- ZIP 仅包含使用 `/` 的文件条目，不含目录条目，兼容手机版设置页导入器。

配置与存档数据不放在 MOD 文件夹内，替换 DLL/整包安装不会清空卡图选择、卡牌禁用状态、动态出率和历史版本升级选择。
