# BetterDefect v0.10.13

适用版本：

- 手机：Slay the Spire 2 v0.103.x
- 电脑：Slay the Spire 2 v0.107.1
- 不需要 BaseLib

## 本次更新

- 新增“冷静头脑”的百科大全改造选项，占用 35 点改造上限中的 1 点：
  - 基础：1费，抽2张牌，然后生成1个冰霜充能球，消耗。
  - 升级：1费，抽2张牌，然后生成1个冰霜充能球，去除消耗。
- 修改“混沌”的改造：
  - 基础：1费，优先生成当前栏位中没有的随机充能球，共生成2个，消耗。
  - 升级：生成数量仍为2个，去除消耗。
- 两张牌的卡面描述、百科改造摘要、实际战斗逻辑和普通升级预览保持一致。
- 原有动态出率、禁用状态、卡面选择和改造记录继续保留。

## 安装

解压后将 `BetterDefect` 文件夹放入游戏的 `mods` 目录，覆盖旧版同名文件夹即可。
