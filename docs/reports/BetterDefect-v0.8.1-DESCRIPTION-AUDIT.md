# BetterDefect v0.8.1 描述一致性审计

审计范围：22 张重建的一代故障机器人卡牌、4 张游戏内隐藏恢复卡牌、14 项历史版本改造，以及这些卡牌关联的自定义能力。

| 项目 | v0.8.0 问题 | v0.8.1 处理 |
|---|---|---|
| 火箭飞拳 | 基础版本实际在回合结束时失去 0 费，文字却始终写“下一次打出前” | 文字随 v0.100 开关切换 |
| 特斯拉线圈 | 未启用 v0.105 时，升级预览仍写闪电触发两次，实际只触发一次 | 只有启用 v0.105 且普通升级后才显示两次 |
| 燃料 / 压缩 | v0.108 基础燃料不抽牌，但显示“抽 0 张牌” | 基础版本隐藏抽牌行；v0.99 显示抽 1/2 张 |
| 刮削 | 基础版本与 v0.108 对临时 0 费牌判断不同，原文字无法区分 | 分别说明“卡牌自身耗能”和“当前最终耗能” |
| 裂变 | 普通与升级行为同时写在一张牌上，并重复显示消耗 | 用 `IfUpgraded` 只显示当前行为；消耗交给真实关键词显示 |
| 核心电涌 | 描述手写消耗，同时真实关键词又会生成消耗提示 | 删除手写重复行 |
| 增幅 | 描述写“本回合”，能力却可跨回合保留 | 玩家回合结束时移除；按原生卡牌次数修改钩子消耗层数 |

其余卡牌的数值使用动态变量渲染；耗能、固有、消耗等由当前真实卡牌属性生成，与普通升级和历史版本状态同步。离线自动检查结果为 66/66 通过。
