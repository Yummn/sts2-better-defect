# BetterDefect v0.9.2

## 修复内容

- 修复一代能力在战斗中角色状态栏显示红色 `NOPE` 占位图的问题。
- 同时修复能力详情大图的缺失图标。
- 为 6 个自定义能力补齐 `smartDescription`，状态栏悬浮说明不再回退或缺字。

## 原因

BetterDefect 的一代能力使用 `BD_*` 模型 ID。游戏会按模型 ID 自动查找能力图标，但 v103/v107.1 的内置图集中没有对应的 `bd_*` 资源，因此加载了 `missing_power.png`（图中显示 `NOPE`）。

## 图标映射

- 散热片 → 集中（游戏未内置散热片图标时的稳定替代）
- 自我修复 → 再生
- 静电释放 → 荆棘
- 扩容 → 爆发
- 电动力学 → 避雷针
- 锁定 → 易伤

映射同时覆盖角色下方的小状态图标和能力详情大图，并在使用前检查资源确实存在。

## 兼容版本

- 手机：Slay the Spire 2 v0.103.2
- 电脑：Slay the Spire 2 v0.107.1
- 不需要 BaseLib
