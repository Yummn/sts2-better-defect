# BetterDefect v0.8.4

适配：手机版 v0.103.2、电脑版 v0.107.1；不依赖 BaseLib。

## 本版修复

- 修复机器人改造点数进度条会在百科大全之外继续显示的问题。
- 修复从卡牌详情或其他子界面返回百科大全后，进度条有时不会重新显示的问题。
- 原因是 `NCardLibrary.OnSubmenuOpened/Closed` 只在百科界面真正入栈/出栈时触发；打开卡牌详情时只是把百科临时隐藏，返回时只是重新显示，因此旧逻辑收不到完整生命周期事件。
- 现在进度条绑定到实际触发显示的 `NCardLibrary` 实例，并监听 `NSubmenu.OnScreenVisibilityChange`：百科隐藏时立即隐藏，重新显示时立即恢复。
- 移除了全局场景树名称扫描，不再根据残留的百科节点猜测显示状态，也降低了手机端额外遍历开销。
- 修复禁用数和升级数发生互换但总点数不变时，进度条标题统计不刷新的小问题。

## 保留功能

- 仅百科大全显示禁用按钮、动态出率、历史版本改造和 35 点进度条。
- 卡图选择、禁用状态、动态出率、历史版本改造记录继续跨重启和覆盖安装保存。
- 保留 v0.8.3 的局内牌组 UI 隔离修复及此前全部卡牌效果修复。

## 安装

- 手机版 v103：`BetterDefect-v0.8.4.zip`
- 电脑版 v107.1：`BetterDefect-v0.8.4-PC-v107.1.zip`
- 解压或导入后，把 `BetterDefect` 文件夹放入游戏 `mods/`。
