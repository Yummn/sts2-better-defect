# BetterDefect v0.9.0

适用版本：

- PC：杀戮尖塔 2 v0.107.1
- 手机：杀戮尖塔 2 v0.103.2
- 不依赖 BaseLib

## 新增：白卡“改造：自定义”

在“百科大全 → 故障机器人”中点击卡牌上方的“改造”按钮启用。启用为绿色，关闭为红色；每项占用 1 点，与既有历史版本改造共用 35 点上限，选择会持久保存。

- **Barrage / 弹幕齐射**：1 费，触发所有充能球被动 1 次；普通升级后触发 2 次。
- **Beam Cell / 光束射线**：0 费，施加 1（2）层锁定，不再造成伤害或施加易伤。
- **Charge Battery / 充电电池**：1 费，获得 6（9）格挡；下回合获得 1 能量并抽 1 张牌。
- **Cold Snap / 寒流**：2 费，造成 12（18）伤害并生成 2 个冰霜球。
- **Go for the Eyes / 眼部攻击**：造成 3（4）伤害，无论敌人是否攻击都施加 1（2）层虚弱。
- **Gunk Up / 黏液喷射**：造成 4（5）伤害 3 次，并将 1 张黏液加入手牌。
- **Leap / 飞跃**：获得 8（11）格挡，打出后本场战斗内变为 0 费。
- **Lightning Rod / 引雷针**：获得 3（6）格挡，立即生成 1 个闪电球，下回合再生成 1 个闪电球。
- **Sweeping Beam / 扫荡射线**：对所有敌人造成 6（9）伤害，抽 1（2）张牌。
- **Uproar / 喧闹**：每段 5（7）伤害；自动打出牌时优先选择当前费用为 2 的可打出攻击牌，没有时回退为随机攻击牌。
- **Recursion / 递归**：激发最左侧充能球 2 次并重新生成同类型充能球；普通升级后变为 0 费。
- **Streamline / 精简改良**：造成 13（18）伤害；每次打出使本场战斗中所有 Streamline 少 1 费。
